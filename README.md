# MCQ Processing Assignment - May 4th

## Description
This Java program is a simple educational tool designed to test a user's knowledge of Java syntax. It displays a multiple-choice question about array declaration and uses conditional logic to evaluate the user's response.

## Features
- Displays a formatted MCQ.
- Accepts user input via the `Scanner` class.
- Case-insensitive input handling (accepts 'a' or 'A').
- Provides the correct answer if the user is incorrect.

## How to Run
1. Ensure you have the Java Development Kit (JDK) installed.
2. Download `MCQProcessor.java`.
3. Run the program.
